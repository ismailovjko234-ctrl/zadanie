# Book Tracker — Трекер прочитанных книг
**Автор:** Исмаилов Арслан Айбекович
**Вариант:** Book Tracker
**Дата сдачи:** 30.04.2026

## Описание
Приложение для ведения списка прочитанных книг с возможностью фильтрации по объему (более 200 страниц) и сохранением данных в JSON.

## Инструкция по запуску
1. Установите PyQt6: `pip install PyQt6`
2. Запустите программу: `python main.py`

## Тестирование
| Тип теста | Данные | Ожидаемый результат |
|-----------|--------|---------------------|
| Позитивный | "Оно", "Кинг", 1100 стр | Книга добавлена в список |
| Негативный | Пустое название | Ошибка: "Заполните поля" |
| Валидация | Буквы в поле "Страницы" | Ошибка: "Введите число" |
