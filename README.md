## Описание программы
Приложение SupportPro предназначено для автоматизации работы службы поддержки. Оно позволяет создавать заявки, сохранять их в локальную базу данных (JSON) и просматривать список активных тикетов.

## Инструкция по запуску
1. Установите Python 3.10 или выше.
2. Установите зависимости: `pip install PyQt6`
3. Запустите программу: `python main.py`

## Тестирование
| Тип теста | Описание | Результат |
|-----------|----------|-----------|
| Позитивный | Заполнение полей и нажатие кнопки | Создание JSON-записи |
| Негативный | Попытка сохранить пустые поля | Окно с предупреждением |
| Граничный | Ввод спецсимволов (!@#$) | Корректное сохранение |

## Архитектура
- `main.py`: Интерфейс пользователя (View).
- `logic.py`: Функции обработки данных (Model).
- `tickets.json`: Хранилище данных.
